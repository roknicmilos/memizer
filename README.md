# memizer

Displays a meme!

## Installation

(Optional) Create a virtual environment:

```bash
python -m venv venv
source venv/bin/activate
```

Install the package:

```bash
pip install memizer
```

## Usage

### CLI command:

```bash
memize
```

### Python code:

```python
from memizer import display_meme

display_meme()
```
